# Directory Discovery Map: quant-engine

* `/src/data/`: Data ingestion scripts (Yahoo Finance, Macro, Alt-Data).
* `/src/execution/`: Multi-agent swarm logic and vectorized backtesting.
* `/src/features/`: Alpha factor generation and cross-sectional scoring.
* `/src/utils/`: Contains `pre_flight.py` and universal encoders.
* `/.templates/`: Agentic code generation boilerplates.
* `/tests/`: Standardized test suites.
