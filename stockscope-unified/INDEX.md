# Directory Discovery Map: stockscope-unified

* `/backend/app/`: Primary API logic, schemas, and models.
* `/backend/app/api/`: Endpoint definitions and route handlers.
* `/backend/app/models/`: SQLModel data models.
* `/frontend/src/`: Next.js frontend source code.
* `/frontend/app/`: Next.js 16 app router components and pages.
* `/utils/`: Shared helper scripts and data conversion utilities.
